# Política de Privacidade

**Última atualização:** 24 de Abril de 2026

A nossa prioridade é a transparência e a segurança dos nossos usuários. Esta política descreve como lidamos com as informações no aplicativo **Zabbix App**.

### 1. Coleta de Dados
O aplicativo **Zabbix App** foi desenvolvido para funcionar sem a necessidade de coletar, armazenar ou transmitir qualquer tipo de dado pessoal ou sensível dos seus usuários.

### 2. Uso de Permissões
Caso o aplicativo solicite permissões técnicas (como acesso à câmera ou armazenamento), elas são utilizadas exclusivamente para o funcionamento local das funcionalidades do app. Nenhum dado resultante dessas permissões é enviado para nossos servidores ou para terceiros.

### 3. Serviços de Terceiros
O aplicativo não utiliza serviços de terceiros (como redes de anúncios, ferramentas de análise) que possam rastrear o comportamento do usuário ou coletar dados identificáveis.

### 4. Segurança
Como não coletamos dados, não há risco de vazamento de informações pessoais através deste aplicativo.

### 5. Alterações nesta Política
Podemos atualizar nossa Política de Privacidade ocasionalmente. Recomenda-se que você revise esta página periodicamente para quaisquer alterações.
